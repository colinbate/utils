cssify Your XPath

Enter an xpath expression, and this tool will convert it to a CSS selector. This is particularly useful if you want to target some specific element on a badly designed page when you are creating user styles. Or just if you want to be very specific about some style that you need to apply to your site.

For example, if you are in the developer panel of Chrome, you can right click an element to Copy XPath.

The XPath for this paragraph is: //*[@id="main"]/section/p[3]
