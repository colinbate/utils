Colin's Utilities

Everyone spends so much time in their browser, why not use tools to make the time more productive or enjoyable? Here are a collection of bookmarklets (aka. favlets) and simple tools you can add to your bookmarks and use whenever needed.

Most of these are pretty geeky, built for my own use, but made available here in case someone else has a need. If you have any problems with them, or want to request other features/tools/bookmarklets, then please use the issue tracker.

Bookmarklets

  • Fit To Window—will resize an image to be the width of your browser window.
  • Oh Behave!—will give any textarea coding abilities (indentation, auto-pairing, etc).
  • De-localhost—allows you to replace 'localhost' in your current URL with a string of your choosing for easy copying.
  • Text Resizer—allows you to increase or decrease the text size just in the currently focused field.
  • RhodeCode PR Enhancements—allows you to create a pull request for an arbitrary changeset.
  • Owl Me—adds fun owls to the page you are on.
  • Find TFS Repo ID—will show you the ID of your TFS Git repository.

Tools

  • CSSify—converts an XPath selector into a CSS selector.
