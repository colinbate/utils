Fit To Window

This bookmarklet can be used to resize an image (usually one which is too big) to fit the width of your browser window.

If the page has only one image, it will resize that one (including if you have opened an image directly in the browser). If there are multiple images, you need to click on the one you want to resize after clicking the bookmarklet.

At the moment only tested on Chrome.
